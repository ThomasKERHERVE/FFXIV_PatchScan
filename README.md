# FFXIV PatchScan

Automatically extracts the useful parts of FFXIV patch notes using Claude AI — job adjustments (PvE & PvP), NPC locations, and major new content. No more reading walls of text.

🌐 **[Open the site](https://thomaskerherve.github.io/FFXIV_PatchScan/)**

## Usage

1. Auto-detect the latest patch or paste a URL manually
2. Enjoy the patch in a different way
